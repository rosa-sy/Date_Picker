# Date_Picker
## date picker on flutter
picking the date easily and beautifully by  create a function then calling : 
```
showDatePicker(
      context: context,
      initialDate: DateTime.now(),
      firstDate: DateTime(2020),
      lastDate: DateTime.now(),
    );
```
